# ちよデイリーフロー 🐥

LINE × Googleカレンダー連携のスケジュール管理Bot

副業・本業・プライベートの予定を **LINEから直感的に管理できる個人用スケジュール管理システム**。

---

# 🎯 目的

副業と本業を両立する中で、予定管理をもっとシンプルにしたいと思い開発。

このシステムでは

* 副業
* 本業（看護師）
* プライベート

の予定を **LINEから一元管理**できるようにしています。

主な特徴

* LINEから自然な文章で予定登録
* Googleカレンダーに自動登録
* カレンダーをまとめて確認
* 重複予定チェック
* 朝・夜の定時通知

---

# ⚙ 使用技術

* Python
* FastAPI
* LINE Messaging API
* Google Calendar API
* APScheduler
* Git / GitHub

---

# 💡 現在の機能

### 📅 予定確認

LINEで以下を送ると予定を表示

```
今日の予定
明日の予定
3/15の予定
```

表示例

```
【明日の予定】

📅 副業
14:00 会議

📅 看護師
16:00 夜勤
```

---

### ✏ 予定登録

自然な文章から日時を抽出

例

```
明日16時 夜勤
```

↓

```
予定名を入力してください
```

↓

```
登録先を選んでください
1 副業
2 看護師
3 プライベート
```

↓

Googleカレンダーへ登録

---

### ⚠ 重複チェック

同じ時間帯に予定がある場合

```
⚠ 同じ時間帯に予定があります
それでも登録しますか？

1 はい
2 キャンセル
```

---

### ⏰ 定時通知

毎日

```
朝6:00
今日の予定
```

```
夜21:00
明日の予定
```

をLINEへ自動送信。
