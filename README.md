# chiyo-dailyflow 🐥

LINE × Google Calendar連携のスケジュール管理Bot。

愛鳥ちよのために、副業と本業を両立するための
個人用タスク＆予定管理システムとして開発中。

---

## 🎯 目的

- 副業・本業・プライベートの予定を一元管理
- LINEから直感的に予定登録
- 重複チェック機能
- 朝・夜の定時通知機能

---

## 🛠 使用技術

- Python
- FastAPI
- LINE Messaging API
- Google Calendar API
- APScheduler

---

## 💡 現在の機能

- 「今日の予定」表示
- 「明日の予定」表示
- 日付指定予定登録
- 重複チェック
- 朝6時に今日の予定を通知
- 夜21時に明日の予定を通知


---

- デプロイによる常時稼働
- データ永続化（JSON / DB）
- 自然言語解析強化
- UI改善
